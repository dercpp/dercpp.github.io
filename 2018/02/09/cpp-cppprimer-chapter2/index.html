基本内置类型

C++ 定义了一套包括算数类型与空类型在内的基本数据类型

算数类型包括：

  • 字符
  • 整型数
  • 布尔值
  • 浮点数
类型 含义 最小尺寸
bool 布尔类型 未定义
char 字符 8位
wchar_t 宽字符 16位
char16_t Unicode字符 16位
char32_t Unicode字符 32位
short 短整型 16位
int 整型 16位
long 长整型 32位
long long 长正型 64位
float 单精度浮点型 6位有效数字
double 双精度浮点型 10位有效数字
long double 扩展精度浮点型 10位有效数字
  • 除去布尔型与扩展的字符型外，其他整型可以划分为带符号的（ signed ）和无符号的（ unsigned ） 默认都是带符号的
    • 带符号的类型首位为符号为，无符号的所有位都用于存储值
  • 把非布尔类型的值赋给布尔类型时，初始值为0则布尔类型结果为 false，否则为 true
  • 浮点数赋值给整型时将作近似处理，舍弃小数部分
  • 给无符号类型的赋值若超出其表示范围，则其值为初始值对无符号类型表示数值总数取模后的余数
  • 给带符号类型的赋值超出其表示范围时，结果是未定义的

  • 若表达式里既有带符号类型又有无符号类型，当带符号类型取值为负时表达式结果会出现异常

  • 字符与字符串字面值

变量

变量组成形式：
类型说明符 变量名;

  • 可以在定义变量时直接为变量赋值

初始化形式：

  • int units = 0;
  • int units = {0};
  • int units{0};
  • int units(0);
  • 使用花括号 {}的初始化形式为列表初始化，此时不能用表示范围大的类型初始化表示范围小的类型，编译器会报错
  • 任何定义在函数体外的变量若无初始化，则将被默认初始化为0
  • 定义在函数体内部的内置类型变量将不被初始化
  • 类对象的默认初始化由类决定

变量声明使得变量名为程序所知，而定义负责创建与名字关联的实体

仅声明变量而不定义：

  • extern int i;

声明并定义：

  • int j;

注：

  • C++ 是一种静态类型语言，在编译阶段检查类型

同一个名字在不同作用域中可能指向不同实体。名字的有效区域始于名字的声明语句，终于语句所在作用域末端。

  • 指针 - 指向另一种类型的复合类型

    • 指针也实现了对其他对象的间接访问
    • 指针本身即是对象，允许对其赋值、拷贝
    • 指针生命周期内可以先后指向几个不同的对象
    • 不是必须在定义时赋初始

    • 指针值（即地址）应属下列四种状态之一：

      • 指向一个对象
      • 指向紧邻对象所占空间的下一个位置
      • 空指针，建议使用 nullptr 来初始化指针得到空指针，尽量避免用 NULL
      • 无效指针（上述情况之外的其他值）

    • 利用指针访问对象

    • void* 指针是一种特殊类型的指针，可以存放任意对象的地址

      • void* 指针作用有限，一般用于与别的指针作比较或作为函数的输入、输出，或给另外一个 void* 指针赋值
      • 因为不知道 void* 指针所指对象的具体类型，故不能直接操作其所指对象
